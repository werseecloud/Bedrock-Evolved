# World Uplift Structure Templates

The pack includes generated low-detail `.mcstructure` files for immediate testing. Replace them with hand-built exports when you want production-quality cities and mountain landmarks.

Export workflow:

1. Build the structure in a creative test world.
2. Use a structure block.
3. Set the name to the listed identifier path, for example `city:town_hall`.
4. Export the `.mcstructure`.
5. Place the file in the matching folder under `behavior_packs/WorldUpliftBP/structures`.
